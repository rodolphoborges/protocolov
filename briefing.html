Selecione um protocolo abaixo para ver as instruções de execução.

/convocar [CÓDIGO] [MISSÃO]

PROTOCOLO: REFORÇO ORBITAL

"Não morra sozinho. Peça ajuda." - Brimstone
01. Crie um Grupo (Party) dentro do Valorant.
02. Copie o código de 6 dígitos gerado pelo jogo.
03. No Telegram, digite /convocar seguido do código.
04. O site exibirá um alerta vermelho para todos os membros.
/unidade

PROTOCOLO: ASCENSÃO DE PATENTE

"A reserva é para os fracos. Escolha um lado." - Viper
01. Ative o comando no Terminal para ver as unidades.
02. Selecione ALPHA (Liderança) ou OMEGA (Assalto).
03. Procure seu nome na lista filtrada por elo.
04. Confirme a transferência para atualizar seu registro no site.
/perfil [RIOT_ID]

PROTOCOLO: ANÁLISE DE DADOS

"Eu sei tudo sobre você. Meus robôs viram." - Killjoy
01. Digite /perfil e uma parte do nome do agente.
02. O sistema extrai: Elo, Sinergia e Pontuação de Treino.
03. Use isso para analisar se o recruta presta para o seu time.
/ranking

PROTOCOLO: HIERARQUIA DE ELITE

"Números não mentem. Você é útil ou é lixo?" - Viper
01. Dispare o comando para ver quem são os agentes ativos.
02. O Bot lista os Top 3 em Sinergia (Trabalho em Equipe).
03. Os nomes aparecem com o emoji de Elo atualizado via API.
